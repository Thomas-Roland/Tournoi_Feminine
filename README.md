# Déploiement GitHub Pages

Pour publier ce site sur GitHub Pages :

1. Allez sur le dépôt GitHub : https://github.com/Thomas-Roland/Tournoi_Feminine
2. Cliquez sur "Settings" (Paramètres).
3. Dans le menu à gauche, cliquez sur "Pages".
4. Dans la section "Branch", choisissez `main` et le dossier `/ (root)`.
5. Cliquez sur "Save".

Votre site sera accessible à l’adresse indiquée par GitHub Pages (ex : https://thomas-roland.github.io/Tournoi_Feminine/)

⚠️ Assurez-vous que vos chemins d’images et de scripts sont relatifs (ex : `img/monimage.png`).
